# Yazi 操作速查

> 配置目录：`~/.config/yazi/`  
> 终端直接跑 `yazi`，与 Neovim 里的 `yazi.nvim` **共用本配置**。  
> **新电脑安装 / 插件清单 → [`SETUP.md`](./SETUP.md)**

---

## 从 Neovim 打开

| 快捷键 | 功能 |
|--------|------|
| `tt` / `-` | 打开 Yazi（当前文件所在目录） |
| `nvim <目录>` | 直接用 Yazi 打开目录（已接管 netrw） |

Neovim 侧绑定见：`~/.config/nvim/KEYMAPS.md` →「文件管理」。

---

## 导航 / 打开

| 键 | 功能 |
|----|------|
| `h` / `l` | 上级目录 / **smart-enter**（进目录或打开文件） |
| `Enter` | 同 `l`（smart-enter） |
| `j` / `k` | 上下移动 |
| `K` | 预览里向上滚（默认能力，视版本而定） |
| `g` `g` / `G` | 跳到列表顶 / 底 |
| `z` | **zoxide** 智能跳目录 |
| `Z` | **fzf** 模糊定位目录/文件 |

### 书签跳转（自定义）

| 键 | 目标 |
|----|------|
| `g` `p` | `~/projects` |
| `g` `D` | `~/Downloads` |
| `g` `C` | `~/.config` |
| `g` `r` | 当前 Git 仓库根目录 |

---

## 选择 / 文件操作

| 键 | 功能 |
|----|------|
| `Space` | 勾选 / 取消勾选 |
| `v` | 可视多选（默认） |
| `y` / `x` | 复制（yank）/ 剪切 |
| `p` | **smart-paste**（悬停目录则贴进去，否则贴到 CWD） |
| `d` | 删除（入回收逻辑，以你本地默认为准） |
| `r` | 重命名 |
| `a` / `A` | 创建文件 / 创建目录（默认能力） |
| `X` | 打开悬停项（压缩包走 `extract` opener） |
| `.` / `<C-h>` | 切换隐藏文件 |

---

## 复制路径 / Shell

| 键 | 功能 |
|----|------|
| `c` `p` | 复制完整路径 |
| `c` `n` | 复制文件名 |
| `!` | 在当前目录打开 `$SHELL`（阻塞） |

---

## Tab / 其它常用默认键

| 键 | 功能 |
|----|------|
| `t` | 新建 tab |
| `[` / `]` | 上/下一个 tab |
| `~` | 回到 `$HOME` |
| `-`（Yazi 内） | 回到上一目录（与 Neovim 的 `-` 打开 Yazi 不是同一层） |
| `f` / `s` / `S` | 过滤 / 按文件名搜 / 按内容搜（依赖 fd、rg） |
| `q` | 退出 |

> Neovim 浮动窗里部分键可能被 `yazi.nvim` 拦走（分屏打开等），按 `<f1>` 看插件帮助。

---

## 已启用插件

完整安装步骤与依赖见 **[`SETUP.md`](./SETUP.md)**。

| 包 | 作用 |
|------|------|
| `smart-enter` | `l`/`Enter` 智能进入或打开 |
| `smart-paste` | `p` 智能粘贴 |
| `full-border` | 圆角全边框 |
| `git` | 列表显示 Git 状态 |
| `catppuccin-mocha` | 主题（flavor） |
| `zoxide` / `fzf`（内置） | `z` / `Z`，需系统已装对应程序 |

新电脑一键：

```bash
# 同步 ~/.config/yazi/ 后
ya pkg install
```
